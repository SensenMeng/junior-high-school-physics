# 迁移到拯救者 Legion 部署指南

## 项目结构总览

```
初中物理/
├── backend/           # FastAPI 后端（含 ingest 导入脚本）
├── frontend/          # React 前端
├── data/              # PDF + Chroma 向量数据库
│   ├── textbooks/     # 三本教材 PDF（八上、八下、九年级）
│   └── chroma_db/     # 向量数据库（当前为百川 API 版本）
├── start.sh           # 启动脚本
└── 迁移拯救者指南.md   # 本文件
```

## 1. 复制项目到 U 盘

```bash
# 先删除前端 node_modules（体积大，到拯救者上重装）
cd /Users/msensen/Code/初中物理
rm -rf frontend/node_modules

# 检查总大小（应该 50MB 左右）
du -sh .

# 把整个 初中物理 文件夹复制到 U 盘
cp -r /Users/msensen/Code/初中物理 /Volumes/你的U盘/
```

## 2. 在拯救者上设置环境

```bash
# 进入项目
cd /path/to/初中物理/backend

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装后端依赖
pip install -r requirements.txt

# 额外安装 BGE 本地模型需要的包
pip install sentence-transformers torch torchvision torchaudio

# 前端依赖
cd ../frontend
npm install
```

## 3. 切换到本地 BGE 模型

```bash
cd /path/to/初中物理/backend
source venv/bin/activate

# 备份原来的 embedding.py（百川 API 版）
mv embedding.py embedding_baichuan.py

# 把 BGE 版本设为默认
cp embedding_bge.py embedding.py
```

> **注意**：BGE 的向量维度和百川 API 不同，已有的章节向量需要**清空重导**。

## 4. 完整重导入（章节 + 句子 + 关键词）

`ingest.py` 已支持 `--full` 参数。使用本地 BGE 不限速，一次性完成三粒度导入。

```bash
cd /path/to/初中物理/backend
source venv/bin/activate

# 第一步：清空旧数据（百川版的向量跟 BGE 不兼容）
rm -rf ../data/chroma_db/
mkdir -p ../data/chroma_db/

# 第二步：重新导入三本教材（用 --full 包含句子和关键词）
# PDF 文件名长，输入时可拖入终端或按 Tab 补全
python3 ingest.py "../data/textbooks/义务教育教科书 物理 八年级 上册 (人民教育出版社 课程教材研究所 物理课程教材研究开发中心) (z-library.sk, 1lib.sk, z-lib.sk).pdf" "人教版物理2024八年级上册" --full
python3 ingest.py "../data/textbooks/义务教育教科书物理八年级下册 (人民教育出版社) (z-library.sk, 1lib.sk, z-lib.sk).pdf" "人教版物理2024八年级下册" --full
python3 ingest.py "../data/textbooks/物理 九年级 全一册 (人民教育出版社, 课程教材研究所, 物理课程教材研究开发中心) (z-library.sk, 1lib.sk, z-lib.sk).pdf" "人教版物理2024九年级全一册" --full
```

## 5. 启动服务

```bash
cd /path/to/初中物理
./start.sh
```

前端访问：http://localhost:5173
后端 API：http://localhost:5173/api

## 注意事项

- Chroma 数据库位置：`data/chroma_db/`，如果导入出错可以直接删掉重来
- 切换到 BGE 后，`backend/.env` 中的 API Key 不再需要，保留即可
- BGE 模型首次运行会从 HuggingFace 下载约 1.3GB，需要联网
- 如果学校网慢，可以在 Mac 上先跑一次 `python3 -c "from sentence_transformers import SentenceTransformer; SentenceTransformer('BAAI/bge-large-zh-v1.5')"` 把模型下好，然后把 `~/.cache/huggingface/` 一起拷到 U 盘
